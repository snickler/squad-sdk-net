# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

Instead, please report them through one of the following channels:

1. **GitHub Security Advisories**: Use the [Security tab](https://github.com/snickler/squad-sdk-net/security/advisories/new) to privately report a vulnerability.
2. **Email**: Contact [@snickler](https://github.com/snickler) directly via the email listed on their [GitHub profile](https://github.com/snickler).

### What to Include

- A description of the vulnerability
- Steps to reproduce or a proof of concept
- The potential impact

### Response Timeline

- **Acknowledgment**: Within 48 hours
- **Initial assessment**: Within 5 business days
- **Fix or mitigation**: Targeted within 30 days, depending on severity

We appreciate your help in keeping Squad.SDK.NET and its users safe.
